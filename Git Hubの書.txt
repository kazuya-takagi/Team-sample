Git Hubの流れ

Git Hub上の該当データをクローンします。

作業を始める前に自分が編集する用のローカルブランチを作成してそのブランチに切り替えます。(ex.  feature/megane)

作業開始

終わったらまずプロジェクトのフォルダごとコミット(リビジョングラフ更新)

(要確認)編集・コミットしたブランチを空欄のままプッシュ。

Git Hubを確認

マージプルリクエスト。承認されたらブランチを削除（Git Hub上では消える）。

ブランチをdevelopにしていない場合は切り替える。

フェッチ&リベースからマージを選択。古いブランチをクリーンアップ(リビジョングラフ更新)。
リビジョングラフ上で編集用ブランチを削除。
